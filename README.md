## Run Locally

Clone the project

```bash
  git clone https://github.com/siambd/nexisltd-task
```

Go to the project directory

```bash
  cd nexisltd-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
